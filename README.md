# Tic Tac Toe Game on Casper Test-Net Network
Tic tac toe game builded on Casper Test-Net Network written in Javascript. 

## Install

```bash
git clone https://github.com/socodes/TicTacToe-on-Casper
cd TicTacToe-on-Casper
npm install
node server.js
```

## Dependencies:
- Javascript
- Rust Language
- CMake
- Casper-client
- axios
- express.js
- cors.js
